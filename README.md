## K E Y B I N D S

<details>
<summary> Mis atajos de teclado generales</summary>

|   Acion   |   atajo   |
|:----------|:----------|
|   Abrir Terminal      |   'Mod + Q'   |
|   Cerrar Ventana      |   'Mod + C'   |
|   Rofi Menu           |   'Mod + R'   |
|   Selected wallpaper  |   'Mod + W'   |
|   PowerMenu           |   'Mod + Esc' |

Cambiar Foco

|   Accion  |   Atajo   |
|:----------|:----------|
| Foco a la izquierda   |   'Mod + arrowleft'   |
| Foco a la Dercha      |   'Mod + arrowrigth'  |
| Foco hacia Arriba     |   'Mod + arrowup'     |
| Foco hacia Abajo      |   'Mod + arrowdown'   |

Moverse en workspaces

|    Acion  | Atajo    |
|:----------|:---------|
|  Desplazarse de 1-9  |  'Mod + 1-9'          |
|  Mover ventanas 1-9  |  'Mod + SHIFT + 1-9'  |

Subir y Bajar Volumen

|  Accion  |  Atajo    |
|:----------|:---------|
|  Subir Volumen       |  'F12'  |
|  Bajar Volumen       |  'F11'  |
</details>
